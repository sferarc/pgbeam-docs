List all saved authentication profiles with their method, organization, and
email. The currently active profile is marked with an asterisk (`*`). If no
profiles are configured, prints instructions to run `pgbeam auth login`.

## Usage

```bash
pgbeam auth list
```

<Callout type="info" title="Global options">
  All [global options](/docs/cli/global-options) (`--token`, `--profile`,
  `--project`, `--org`, `--json`, `--no-color`, `--debug`) are also available on
  this command.
</Callout>

## Examples

```bash
# List all profiles
pgbeam auth list

# List profiles as JSON
pgbeam auth list --json
```

## Output

Displays a table with columns: active indicator, profile name, auth method,
organization ID, and email. With `--json`, returns an array of profile objects.
