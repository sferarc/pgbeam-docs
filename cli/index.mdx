---
title: Installation
description: Install the PgBeam CLI on macOS or Linux.
---

<Tabs items={["macOS", "Linux"]}>
  <Tab value="macOS">
    ```bash
    curl -fsSL https://pgbeam.com/install | sh
    ```

    The installer downloads a precompiled binary and adds it to your PATH.

  </Tab>

  <Tab value="Linux">
    ```bash
    curl -fsSL https://pgbeam.com/install | sh
    ```

    Supports x86\_64 and ARM64 architectures.

  </Tab>
</Tabs>

Verify the installation:

```bash
pgbeam --version
```

After installing, authenticate with your PgBeam account:

```bash
pgbeam auth login
```

See [auth login](/docs/cli/auth/login) for details.
