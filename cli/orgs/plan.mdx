---
title: orgs plan
description: Show current plan details and limits.
---

Show the current plan details and resource limits for the active organization.

## Usage

```bash
pgbeam orgs plan
```

## Examples

```bash
pgbeam orgs plan
# Organization: my-org
# Plan:         Pro
#
# RESOURCE        LIMIT
# Databases       10
# Connections     10,000
# Bandwidth       100 GB / month
# Regions         5
# Members         25
```
