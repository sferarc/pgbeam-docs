---
title: orgs usage
description: Show organization usage metrics.
---

Show current usage metrics for the active organization, including connections, queries, and bandwidth.

## Usage

```bash
pgbeam orgs usage
```

## Examples

```bash
pgbeam orgs usage
# Organization: my-org
# Billing period: Mar 1 – Mar 31, 2026
#
# METRIC          USED       LIMIT
# Connections     1,240      10,000
# Queries         482K       unlimited
# Databases       3          10
# Bandwidth       12.4 GB    100 GB
```
