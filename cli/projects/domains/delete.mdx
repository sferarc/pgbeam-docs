---
title: domains delete
description: Remove a custom domain.
---

Remove a custom domain from a project. This revokes the associated TLS certificate.

## Usage

```bash
pgbeam domains delete <domain>
```

Alias: `pgbeam domains rm`

## Examples

```bash
# Remove a custom domain
pgbeam domains delete db.example.com

# Using the alias
pgbeam domains rm db.example.com
```
