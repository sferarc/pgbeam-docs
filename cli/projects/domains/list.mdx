List all custom domains configured for the linked project. Shows each domain's
ID, hostname, verification status, and creation date.

## Usage

```bash
pgbeam domains list
```

<Callout type="info" title="Global options">
  All [global options](/docs/cli/global-options) (`--token`, `--profile`,
  `--project`, `--org`, `--json`, `--no-color`, `--debug`) are also available on
  this command.
</Callout>

## Examples

```bash
# List all custom domains
pgbeam domains list

# List domains as JSON
pgbeam domains list --json
```

## Output

Displays a table with columns: ID, Domain, Verified (yes/no), and Created. With
`--json`, returns the full domain list from the API.
