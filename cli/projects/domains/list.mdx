List custom domains and their verification status.

## Usage

```bash
pgbeam domains list
```

Alias: `pgbeam domains ls`

## Examples

```bash
pgbeam domains ls
# DOMAIN              STATUS     CERTIFICATE
# db.example.com      verified   active
# api.example.com     pending    provisioning
```
