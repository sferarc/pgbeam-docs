List all projects in your organization.

## Usage

```bash
pgbeam projects list [flags]
```

Alias: `pgbeam projects ls`

## Flags

| Flag | Description |
| ---- | ----------- |
| `--json` | Output as JSON |

## Examples

```bash
# List projects
pgbeam projects list

# Using the alias
pgbeam projects ls

# Output as JSON for scripting
pgbeam projects list --json
```
